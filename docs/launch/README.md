| File | Platform | Best time to post |
|---|---|---|
| [show-hn.md](./show-hn.md) | Hacker News | Tue–Thu, 8–10am PT (peak US engineers) |
| [twitter.md](./twitter.md) | X / Twitter | Tue–Thu, evening US time |
| [reddit.md](./reddit.md) | r/ClaudeAI (also r/programming, r/commandline) | weekday afternoons |
| [v2ex.md](./v2ex.md) | V2EX / 即刻 / 小红书 (中文社区) | 工作日晚上 |

## Tips for each platform

- **HN**: don't over-promise. Title is just the project + 1-line tag. Be ready to answer questions calmly.
- **X**: lead with the GIF, not a wall of text. Pin the thread on your profile for a week.
- **Reddit**: include real screenshots/diagrams. Mods dislike pure self-promotion — make the post useful even if no one installs it.
- **中文社区**: 把 "claude code 跨设备" 这个具体痛点放在标题里，国人最关心账号在不在、能不能正常用、官方支不支持。
